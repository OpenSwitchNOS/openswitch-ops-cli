## Validation of traceroute IPv4 parameters ##
### **Objective**  ##
Verify that valid IPv4 address and valid value for optional parameters is passed by the user.
### Requirements ###
The requirements for this test case are:

 - OpenSwitch OS
### Setup ###

#### Topology Diagram ####
```ditaa

    +-------+
    |       |
    |  DUT  |
    |       |
    +-------+

```
#### Test Setup ####
- 1 DUT in standalone

### Description ###
DUT must be running OpenSwitch OS to execute this test, should be by default configured and in the login or bash-shell context.
### Test Result Criteria ###

#### Test Pass Criteria ####
Appropriate Error message is displayed when Invalid value is passed.
#### Test Fail Criteria ####
User is able to pass invalid values.
